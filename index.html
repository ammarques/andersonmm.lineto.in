What I'm currently working on:

Restaurant Finder This is going to be the base of many projects that I have in mind... and a soon to be NodeJS module; my goal here is to be able to find my way to Restaurants public data available in websites like OpenTable, Yelp, TripAdvisor and many others without using their official API's... why you may ask? because they're not available for simple mortals like myself.

It's basicaly a scraper using Puppeteer and afinity text analysis in order to search, classify and provide a JSON with restaurant suggestions that normaly wouldn't be available without access to their official API's.

https://github.com/ammarques/restaurant-finder
Text Analysis My goals here is to grab a piece of text and squeeze information out of it by performing text analysis like detecting sentiment, extracting entities and key word highlighting.

My stack here so far is a Linux Centos 7, Docker container with the amazing Stanford NLP project in Java, Google Cloud NLP and AFINN table library for NodeJS for storage I'm considering MongoDB.
